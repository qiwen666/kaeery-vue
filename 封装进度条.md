封装进度条

| 参数        | 说明                 | 类型    | 默认值  |
| ----------- | -------------------- | ------- | ------- |
| step        | 进度条累加器         | Number  | 1       |
| stepList    | 进度条显示文本       | Array[] | -       |
| activeColor | 激活状态时进度条样式 | String  | #409EFF |
| circleWidth | 圆宽度               | Number  | 80      |
| lineWidth   | 线宽度               | Number  | 90      |
